Controladores

Aprende a crear y gestionar controladores para manejar la lógica de tu aplicación de manera eficiente.

Conceptos Básicos

Estructura MVC

Los controladores son parte fundamental del patrón MVC, manejando la lógica de negocio.

Acciones

Métodos que responden a las peticiones HTTP y procesan la lógica de la aplicación.

Creación de Controladores

1. Controlador Básico

namespace App\Controllers;

class UserController extends Controller {
    public function index() {
        $users = User::all();
        return view('users.index', ['users' => $users]);
    }

    public function show($id) {
        $user = User::find($id);
        return view('users.show', ['user' => $user]);
    }
}

2. Controlador con Validación

public function store() {
    $validated = request()->validate([
        'name' => 'required|min:3',
        'email' => 'required|email|unique:users',
        'password' => 'required|min:8'
    ]);

    $user = User::create($validated);
    return redirect()->route('users.show', $user->id);
}

Características Avanzadas

Middleware en Controladores
class AdminController extends Controller {
    public function __construct() {
        $this->middleware('auth');
        $this->middleware('admin');
    }

    public function dashboard() {
        // Solo accesible para administradores
    }
}
Inyección de Dependencias
class OrderController extends Controller {
    private $orderService;

    public function __construct(OrderService $orderService) {
        $this->orderService = $orderService;
    }

    public function process() {
        return $this->orderService->process();
    }
}

Buenas Prácticas

Principio de Responsabilidad Única

class UserController extends Controller {
    private $userRepository;
    private $mailService;

    public function __construct(
        UserRepository $userRepository,
        MailService $mailService
    ) {
        $this->userRepository = $userRepository;
        $this->mailService = $mailService;
    }

    public function register() {
        $user = $this->userRepository->create(
            request()->validated()
        );

        $this->mailService->sendWelcomeEmail($user);

        return redirect('/dashboard');
    }
}
