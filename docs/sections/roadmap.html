Descubre el futuro del Framework Áureo y las características planificadas para próximas versiones.
v2.0.0 Planificado

Nueva Arquitectura

  • Rediseño completo del núcleo del framework
  • Soporte para PHP 8.2
  • Sistema de módulos mejorado
  • Nueva API de eventos

Previsto para Q3 2024

v1.5.0 En Progreso

Mejoras de Rendimiento

  • Sistema de caché distribuido
  • Optimización de consultas ORM
  • Lazy loading mejorado
  • Compresión automática de assets

Características de Seguridad

  • Autenticación de dos factores

  • Auditoría de seguridad automática

Herramientas de Desarrollo
