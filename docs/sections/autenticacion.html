Autenticación

Aprende a implementar un sistema de autenticación seguro en tu aplicación usando el Framework Áureo. Esta guía te enseñará:
  • Diferentes métodos de autenticación disponibles
  • Implementación paso a paso de login y registro
  • Protección de rutas y recursos
  • Gestión de sesiones y tokens
  • Recuperación de contraseñas y verificación de email

Métodos de Autenticación

Autenticación Tradicional

Sistema de login basado en sesiones con soporte para recordar usuario.

JWT

Autenticación mediante tokens JWT para APIs y aplicaciones SPA.

Importante: Asegúrate de configurar correctamente las variables de entorno relacionadas con la seguridad.

Implementación Básica

1. Configurar el Middleware de Autenticación

// app/Middleware/AuthMiddleware.php
namespace App\Middleware;

class AuthMiddleware {
    public function handle($request, $next) {
        if (!auth()->check()) {
            return redirect('/login');
        }
        return $next($request);
    }
}

2. Implementar el Controlador de Autenticación

// app/Controllers/AuthController.php
namespace App\Controllers;

class AuthController extends Controller {
    public function login() {
        $credentials = request()->only(['email', 'password']);
        
        if (auth()->attempt($credentials)) {
            return redirect('/dashboard');
        }
        
        return back()->with('error', 'Credenciales inválidas');
    }

    public function logout() {
        auth()->logout();
        return redirect('/login');
    }
}

3. Definir las Rutas

// routes/web.php
Router::get('/login', 'AuthController@showLogin');
Router::post('/login', 'AuthController@login');
Router::post('/logout', 'AuthController@logout');

Router::group(['middleware' => 'auth'], function() {
    Router::get('/dashboard', 'DashboardController@index');
});

Características Avanzadas

Roles y Permisos

Sistema flexible de control de acceso basado en roles.

if (auth()->user()->hasRole('admin')) {
    // Acceso a funcionalidades de administrador
}
Autenticación de Dos Factores

Añade una capa extra de seguridad a tu aplicación.

auth()->require2FA();
