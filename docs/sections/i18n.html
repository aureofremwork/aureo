Internacionalización (i18n)

Aprende a crear aplicaciones multilingües con el sistema de internacionalización del Framework Áureo.

Configuración Básica

Configuración de Idiomas
// config/i18n.php
return [
    'default' => 'es',
    'fallback' => 'en',
    'supported' => ['es', 'en', 'fr'],
    'detect_from_browser' => true,
];
Estructura de Archivos
resources/
  lang/
    es/
      messages.php
      validation.php
    en/
      messages.php
      validation.php

Uso en las Vistas

Traducciones Básicas


{{ __('messages.welcome') }}
Pluralización

'items' => '{0} No hay items|{1} Un item|[2,*] :count items'


{{ trans_choice('messages.items', $count) }}

Cambio de Idioma

Middleware de Idioma

class LanguageMiddleware
{
    public function handle($request, $next)
    {
        if ($request->has('lang')) {
            app()->setLocale($request->lang);
            session(['locale' => $request->lang]);
        } elseif (session('locale')) {
            app()->setLocale(session('locale'));
        }

        return $next($request);
    }
}

Validación Multilingüe

// resources/lang/es/validation.php
return [
    'required' => 'El campo :attribute es obligatorio.',
    'email' => 'El campo :attribute debe ser un correo válido.',
    'min' => [
        'numeric' => 'El campo :attribute debe ser al menos :min.',
        'string' => 'El campo :attribute debe tener al menos :min caracteres.'
    ]
];

// Uso en el controlador
$validator = Validator::make($request->all(), [
    'email' => 'required|email',
    'password' => 'required|min:8'
]);
