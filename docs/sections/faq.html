🚀 Primeros Pasos

¿Qué necesito para empezar con el Framework Áureo?

Para comenzar a crear aplicaciones con el Framework Áureo, necesitarás:

  • Un ordenador con cualquier sistema operativo (Windows, Mac o Linux)
  • PHP 8.0 o superior instalado en tu sistema
  • Las extensiones básicas de PHP: PDO, JSON, y OpenSSL
  • Composer instalado (te explicamos cómo instalarlo más abajo)
  • Un editor de código (recomendamos Visual Studio Code)
¿Nuevo en programación? No te preocupes, te guiaremos paso a paso en la instalación de todas las herramientas necesarias.
¿Cómo instalo el Framework Áureo?

La instalación es muy sencilla. Sigue estos pasos:

  1. Abre tu terminal o línea de comandos
  2. Navega hasta la carpeta donde quieres crear tu proyecto
  3. Ejecuta el siguiente comando:
composer create-project aureo/framework mi-proyecto

Este comando creará una nueva carpeta llamada "mi-proyecto" con todo lo necesario para empezar.

¡Listo! Tu proyecto ya está configurado con AdminLTE como plantilla por defecto.

¿Cómo manejo los errores y mensajes para el usuario?

El Framework Áureo hace que el manejo de errores sea sencillo y amigable para el usuario. Aquí te mostramos cómo:

1. Mostrar mensajes al usuario
// En tu controlador
public function guardar() {
    try {
        // Tu lógica aquí
        flash('¡Datos guardados correctamente!', 'success');
    } catch (Exception $e) {
        flash('Hubo un error al guardar los datos', 'error');
    }
    return redirect('/dashboard');
}
2. Validación de formularios
public function registro() {
    $validacion = validate($_POST, [
        'nombre' => 'required|min:3',
        'email' => 'required|email',
        'password' => 'required|min:6'
    ]);

    if ($validacion->falla()) {
        // Muestra los errores automáticamente en la vista
        return back()->withErrors($validacion);
    }

    // Continúa con el registro
}
Consejo: Usa la función helper flash() para mostrar mensajes temporales al usuario después de una acción.

⚡ Optimización y Rendimiento

¿Cómo puedo hacer que mi aplicación sea más rápida?

El Framework Áureo incluye varias herramientas para optimizar el rendimiento de tu aplicación. Aquí te explicamos las más importantes:

1. Caché de Datos
// Guardar datos en caché
cache_set('productos', $productos, 3600); // 1 hora

// Obtener datos de caché
$productos = cache_get('productos');

// Caché con función de respaldo
$productos = cache_remember('productos', 3600, function() {
    return db_query('SELECT * FROM productos');
});
2. Optimización de Consultas
// Consulta optimizada con paginación
$usuarios = db_paginate('usuarios', 10); // 10 usuarios por página

// Consulta con relaciones (evita consultas N+1)
$pedidos = db_query('pedidos')
    ->with('cliente')
    ->with('productos')
    ->get();
Consejo: AdminLTE está optimizado por defecto. Sus archivos CSS y JS ya están minificados para mejor rendimiento.
3. Compresión de Imágenes
// Subir y optimizar una imagen
$imagen = upload_image($_FILES['imagen'], [
    'width' => 800,
    'height' => 600,
    'quality' => 80
]);
Buena Práctica: Utiliza las herramientas de desarrollo del navegador para medir el rendimiento de tu aplicación.

🌐 Publicación de tu Aplicación

¿Cómo publico mi aplicación en Internet?

Publicar tu aplicación es más fácil de lo que piensas. Te guiaremos paso a paso:

1. Preparar tu Aplicación
# Optimizar la aplicación
php aureo optimize

# Verificar que todo funciona
php aureo test
2. Configurar el Entorno

Crea o edita el archivo .env en la raíz de tu proyecto:

# Configuración básica
APP_NAME="Mi Aplicación"
APP_ENV=production
APP_DEBUG=false
APP_URL=https://tudominio.com

# Base de datos
DB_HOST=localhost
DB_NAME=mi_base_datos
DB_USER=usuario
DB_PASS=contraseña
¡Importante! Nunca compartas tu archivo .env ni lo subas a repositorios públicos.
3. Subir los Archivos

Sube estos archivos y carpetas a tu servidor:

  • ✅ Toda la carpeta app
  • ✅ Toda la carpeta public
  • ✅ Toda la carpeta system
  • ✅ El archivo .env
  • ❌ NO subas la carpeta vendor (la recrearemos)
4. Configurar el Servidor

Si usas Apache (el más común), crea o edita el archivo .htaccess:

# Archivo .htaccess
RewriteEngine On
RewriteBase /
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ index.php/$1 [L]
Consejo: AdminLTE ya está configurado y optimizado para producción. No necesitas hacer nada especial con los assets.
5. Últimos Pasos

En tu servidor, ejecuta:

# Instalar dependencias
composer install --no-dev

# Ejecutar migraciones
php aureo migrate

# Limpiar caché
php aureo cache:clear
¡Listo! Tu aplicación ya está en línea. Visita tu dominio para verla en acción.
