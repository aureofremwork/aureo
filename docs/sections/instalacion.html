Instalación

Aprende a instalar y configurar el Framework Áureo en tu entorno de desarrollo. Esta guía detallada te ayudará a:
  • Preparar tu entorno de desarrollo
  • Instalar todas las dependencias necesarias
  • Configurar tu primera aplicación
  • Verificar que todo funciona correctamente
  • Resolver problemas comunes de instalación

Requisitos del Sistema

PHP

PHP 8.0 o superior

Base de Datos

MySQL 5.7+ o MariaDB 10.2+

Extensiones PHP

PDO, JSON, OpenSSL, Mbstring

Herramientas

Composer (Gestor de dependencias)

Pasos de Instalación

1. Crear un Nuevo Proyecto

Abre tu terminal y ejecuta el siguiente comando para crear un nuevo proyecto:

composer create-project aureo/framework mi-proyecto

2. Configurar el Entorno

Copia el archivo de configuración de ejemplo y personalízalo según tus necesidades:

cd mi-proyecto
cp .env.example .env

Edita el archivo .env con tu configuración de base de datos y otras variables de entorno:

DB_HOST=localhost
DB_DATABASE=mi_base_de_datos
DB_USERNAME=usuario
DB_PASSWORD=contraseña

APP_NAME=MiAplicacion
APP_ENV=development
APP_DEBUG=true

3. Instalar Dependencias

Instala todas las dependencias necesarias con Composer:

composer install

4. Configurar Permisos

Asegúrate de que los directorios de almacenamiento tengan los permisos correctos:

chmod -R 775 storage/
chmod -R 775 public/assets/

5. Generar Clave de Aplicación

Genera una clave única para tu aplicación:

php aureo key:generate

6. Ejecutar Migraciones

Configura la base de datos ejecutando las migraciones:

php aureo migrate

7. Iniciar el Servidor de Desarrollo

Inicia el servidor de desarrollo incorporado:

php aureo serve

Tu aplicación estará disponible en http://localhost:8000

¡Felicitaciones! Has instalado correctamente el Framework Áureo. Ahora puedes comenzar a desarrollar tu aplicación.
