Gestión de Archivos

Aprende a gestionar archivos de manera eficiente y segura con el Framework Áureo.

Subida de Archivos

Subida Básica
public function store()
{
    $file = request()->file('documento');
    
    if ($file->isValid()) {
        $path = $file->store('uploads');
        return response()->json([
            'path' => $path
        ]);
    }
}
Validación de Archivos
$request->validate([
    'imagen' => [
        'required',
        'file',
        'image',
        'max:2048',
        'mimes:jpeg,png,jpg'
    ]
]);

Almacenamiento

Configuración de Almacenamiento

// config/storage.php
return [
    'default' => 'local',
    
    'disks' => [
        'local' => [
            'driver' => 'local',
            'root' => storage_path('app')
        ],
        
        's3' => [
            'driver' => 's3',
            'key' => env('AWS_ACCESS_KEY_ID'),
            'secret' => env('AWS_SECRET_ACCESS_KEY'),
            'region' => env('AWS_DEFAULT_REGION'),
            'bucket' => env('AWS_BUCKET')
        ]
    ]
];

Uso del Sistema de Archivos

// Guardar un archivo
$path = Storage::disk('local')->put('archivo.txt', 'Contenido');

// Obtener contenido
$contenido = Storage::get('archivo.txt');

// Verificar existencia
if (Storage::exists('archivo.txt')) {
    // El archivo existe
}

// Eliminar archivo
Storage::delete('archivo.txt');

Manipulación de Imágenes

Redimensionar Imágenes
use System\Image;

$image = Image::make($request->file('imagen'));
$image->resize(300, 200);
$image->save(storage_path('app/public/thumb.jpg'));
Aplicar Filtros
$image = Image::make($path);
$image->brightness(-15);
$image->contrast(10);
$image->greyscale();
$image->save();

Descarga de Archivos

public function download($id)
{
    $documento = Documento::findOrFail($id);
    
    if (Storage::exists($documento->path)) {
        return Storage::download(
            $documento->path,
            $documento->nombre_original
        );
    }
    
    return response()->json([
        'error' => 'Archivo no encontrado'
    ], 404);
}
