Tareas Programadas

Aprende a automatizar tareas y procesos con el sistema de programación de tareas del Framework Áureo.

Configuración Básica

Configuración del Scheduler
// config/schedule.php
return [
    'enabled' => true,
    'timezone' => 'America/Mexico_City',
    'log_output' => true,
    'log_errors' => true
];
Configuración del Cron
# Añadir al crontab
* * * * * cd /ruta/app && php aureo schedule:run >> /dev/null 2>&1

Definir Tareas

Crear una Tarea Programada

namespace App\Tasks;

class EnviarReportesDiarios extends Task
{
    public function handle()
    {
        $stats = Estadisticas::getDiarias();
        Mail::to('admin@ejemplo.com')
            ->send(new ReporteDiario($stats));
        
        $this->info('Reporte diario enviado');
    }
}

Programar la Tarea

// app/Console/Kernel.php
protected function schedule(Schedule $schedule)
{
    $schedule->task(EnviarReportesDiarios::class)
             ->daily()
             ->at('08:00')
             ->withoutOverlapping();

    $schedule->command('backup:run')
             ->weekly()
             ->sundays()
             ->at('01:00');

    $schedule->job(new ProcesarCola)
             ->everyFiveMinutes()
             ->onFailure(function () {
                 Log::error('Fallo al procesar cola');
             });
}

Frecuencias Disponibles

Método Descripción Ejemplo
->daily() Ejecutar diariamente ->daily()->at('13:00')
->weekly() Ejecutar semanalmente ->weekly()->mondays()
->monthly() Ejecutar mensualmente ->monthly()->on(1)
->cron() Expresión cron personalizada ->cron('0 0 * * *')

Características Avanzadas

Condiciones de Ejecución
$schedule->command('emails:send')
         ->daily()
         ->when(function () {
             return Cache::has('send-emails');
         })
         ->skip(function () {
             return App::isDownForMaintenance();
         });
Manejo de Salida
$schedule->command('backup:run')
         ->daily()
         ->appendOutputTo(storage_path('logs/backup.log'))
         ->emailOutputOnFailure('admin@ejemplo.com');

Monitoreo de Tareas

// Verificar estado de tareas
php aureo schedule:list

// Ver log de ejecución
php aureo schedule:log

// Ejecutar tarea manualmente
php aureo schedule:run --task=EnviarReportesDiarios
