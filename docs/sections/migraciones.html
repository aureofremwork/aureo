Migraciones

Aprende a gestionar la estructura de tu base de datos de manera eficiente y controlada con las migraciones.

Conceptos Básicos

¿Qué son las Migraciones?

Las migraciones son como un control de versiones para tu base de datos, permitiendo evolucionar el esquema de forma controlada.

Herramientas

El Framework Áureo proporciona herramientas CLI para crear y ejecutar migraciones fácilmente.

Crear Migraciones

1. Crear una Nueva Migración

php aureo make:migration create_users_table

Esto creará un nuevo archivo de migración en el directorio database/migrations.

2. Estructura de una Migración

use System\Migration;

class CreateUsersTable extends Migration
{
    public function up()
    {
        $this->schema->create('users', function($table) {
            $table->id();
            $table->string('name');
            $table->string('email')->unique();
            $table->string('password');
            $table->timestamps();
        });
    }

    public function down()
    {
        $this->schema->dropIfExists('users');
    }
}

Tipos de Columnas

Tipos Básicos

$table->text('descripcion');

$table->date('fecha');
$table->decimal('precio', 8, 2);
Modificadores
$table->string('email')->unique();
$table->integer('puntos')->default(0);
$table->timestamp('ultima_visita')->nullable();
$table->enum('nivel', ['principiante', 'intermedio', 'avanzado']);

Ejecutar Migraciones

Comandos Principales

# Ejecutar todas las migraciones pendientes
php aureo migrate

# Revertir la última migración
php aureo migrate:rollback

# Revertir todas las migraciones
php aureo migrate:reset

# Revertir y volver a ejecutar todas las migraciones
php aureo migrate:refresh

Relaciones y Claves Foráneas

class CreatePostsTable extends Migration
{
    public function up()
    {
        $this->schema->create('posts', function($table) {
            $table->id();
            $table->string('titulo');
            $table->text('contenido');
            $table->foreignId('user_id')
                  ->constrained()
                  ->onDelete('cascade');
            $table->timestamps();
        });
    }

    public function down()
    {
        $this->schema->dropIfExists('posts');
    }
}
